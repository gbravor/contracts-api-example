### Contracts API Example

## Setup

- Install required engines with `nvm install`
- Install dependencies with `npm install`
- Run DB with `docker-compose up -d`

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
